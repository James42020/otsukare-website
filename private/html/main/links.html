Resources and References

This is a list of resources I have used along my learning journey so far that I think are pretty cool :)


Renshuu

The first learning app I used, great for Hiragana and Katakana, also includes lessons on grammer, JLPT, and popular textbooks, overall I would recommend trying it to see if it works for you


NHK World Japan

Their Learn Japanese feature is great for going over basics, each lesson goes over various diferent topics/situations and enphasises different parts of Japanese such as Kanji, Vocabulary, and even CultureTheir Learn Japanese feature is great for going over basics, each lesson goes over various diferent topics/situations and enphasises different parts of Japanese such as Kanji, Vocabulary, and even Culture


Anki

Arguably the best resource for learning, you can create flashcards, design and add infomation to them as you see fit, they also have community/public flashcards which is what I use for learning, would 100% say to get Anki to help your learning


WaniKana

A great app that I used for first learning Kanji, it puts enphasis on learning radicals which helps out a ton, to this day by learning the radicals it makes memorising kanji easier


KanjiVG

For learning to write Kanji this is a godsend, it turns kanji in SVGs but the viewer can be used to show you how to draw Kanji, and it also links to projects which use it which are also super useful, would 100% recommend


Conjugaizen

This is a new one I have found, but it is very cool, similarly to this website, the creator designed it to learn Japanese so it is very centered around being efficent and has no ads or paywall, a very beautiful learning app, check it out to see if it works for you


Jisho

This is the best Japanese dictionary for searching up words, you can search both in English or in Japanese and has great filters allowing you to search for exactly the words you want


基本 KIHON

Now this might be a little bit narcissistic because this is one of my own projects, but this is my English to Japanese translator, using google translate and linked with Jisho, it allows you to break down any sentence and create word lists, give it a go!


Kanji ah4ve

This is a study tool designed to help you with reading and writing kanji, I found through trying to research radicals which they have a very extensive page on, but i'd recommend checking out the whole website to see if it is good for you


Tofugu

Tofugu is a great resource for reading articles about Japanese language and culture, they are also the creators behind WaniKana so I'd highly recommend checking them out and reading and use the infomation and resources they've got
