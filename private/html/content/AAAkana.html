Kana Description

Hiragana

Hiragana（平仮名）is used for native Japanese words along side kanji, grammer elements like particles, and Furigana, which is the Hiragana that appears above some kanji as a sounding aid

Katakana

Katakana（片仮名）is mainly used for imported words such as テレビ which means television, emphasis similar to typing in all caps, and onomatopoeia which can be used alot in Japanese

These tables have the same 5 collums and 10 rows, and the name for the table in Japanese is 五ご十じゅう音おん which translates to "50 sounds", however 5 of these sound (the ones highlighted in blue) are no longer in use so there is actually only 45 sounds in the 五十音 table.

You may also notice that I said that the table has 10 rows, even though these table actually has 11 rows, This is because the "N" row (and the ん／ン kana) aren't actually part of the orignal 五十音 table but since it is used in Japanese a lot it tends to be added to the 五十音 table as its own row, totaling 46 basic sounds used in the Japanese language.
